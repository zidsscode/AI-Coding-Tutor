# AI Coding Tutor 🤖🐍

## Overview

AI Coding Tutor is an AI-powered terminal application developed using Python and the Google Gemini API. It helps beginners learn Python programming by answering coding questions, explaining code, debugging errors, and providing theory explanations.

This project was built as an internship project to demonstrate the practical use of Generative AI in education.

---

## Features

* 💻 Ask Coding Questions
* 📖 Explain Python Code
* 🐞 Debug Python Code
* 📚 Theory Question Answering
* ⚠️ Error Handling
* 🧩 Modular Python Project Structure

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/AI_Coding_Tutor.git
```

2. Open the project

```bash
cd AI_Coding_Tutor
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Create a `.env` file and add your Gemini API key

```env
GEMINI_API_KEY=your_api_key_here
```

5. Run the project

```bash
python main.py
```
